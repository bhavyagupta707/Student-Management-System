# Student-Management-System
Student Management System built using Python. It allows users to add, view, search, and delete student records through a simple menu-driven interface.  This is a good beginner project. For your CV, I recommend a more advanced version that saves data in a file (CSV/JSON) and includes update functionality.


## Features
- Add Student
- View Students
- Search Student
- Delete Student

## Technologies Used
- Python

## How to Run

1. Clone the repository
2. Open terminal
3. Run:

```bash
python student_management.py
